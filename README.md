# Currículo Web Responsivo

Repositório criado para a entrega do **2º Exercício de Coding - Equipe (VERTICALIZA - 2026.1)**.

## Dados da atividade
- **Curso:** Tecnólogo em Análise e Desenvolvimento de Sistemas  
- **Disciplina:** Coding: Algoritmos e Estrutura de Dados  
- **Professor:** Danilo Farias  
- **Atividade:** Currículo Pessoal em HTML, CSS e Framework de Estilo  

## Objetivo
Desenvolver uma página web representando um currículo pessoal, aplicando os conhecimentos introdutórios de:
- estruturação de conteúdo em HTML
- personalização visual com CSS
- uso de framework para estilização
- responsividade (desktop e mobile)

## Tecnologias utilizadas
- **HTML5**
- **CSS3**
- **Bootstrap 5**

## Estrutura do currículo
A página foi organizada com as seções solicitadas no enunciado:

- **Dados pessoais**
  - nome completo
  - foto
  - cidade/estado
  - e-mail
  - telefone/WhatsApp

- **Objetivo profissional**
  - texto com área de interesse e objetivo

- **Formação acadêmica**
  - curso, instituição e ano de conclusão

- **Conhecimentos / habilidades**
  - linguagens, ferramentas e tecnologias

- **Projetos acadêmicos**
  - descrição breve de atividades realizadas

- **Redes / contato profissional**
  - LinkedIn
  - GitHub
  - Portfólio (quando aplicável)
